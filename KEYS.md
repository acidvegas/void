# TMUX
| Key                  | Description                   |
| -------------------- | ----------------------------- |
| `C + k`              | Kill tab                      |
| `C + n`              | New tab                       |
| `C + t`              | Name tab                      |
| `C + A + t`          | Name session                  |
| `C + A + Left/Right` | Previous/Next session         |
| `C + Left/Right`     | Previous/Next tab             |
| `C + S + Left/Right` | Move tab forwards/backwards   |
| `C + Down/Up`        | Horizontal/Veritcal pane      |
| `C + S + Up/Down`    | Previous/Next pane            |
