# TMUX

###### Sessions
| Key                              | Description                   |
| -------------------------------- | ----------------------------- |
| `Ctrl + Alt + k`                 | Kill session                  |
| `Ctrl + Alt + n`                 | New session                   |
| `Ctrl + Alt + t`                 | Name session                  |
| `Ctrl + Alt + s`                 | Session menu                  |
| `Ctrl + Alt + Left/Right`        | Previous/Next session         |

###### Tabs
| Key                              | Description                   |
| -------------------------------- | ----------------------------- |
| `Ctrl + k`                       | Kill tab                      |
| `Ctrl + n`                       | New tab                       |
| `Ctrl + t`                       | Name tab                      |
| `Ctrl + Left/Right`              | Previous/Next tab             |
| `Ctrl + Shift + Left/Right`      | Move tab forwards/backwards   |

###### Panes
| Key                              | Description                   |
| -------------------------------- | ----------------------------- |
| `Ctrl + Down/Up`                 | Horizontal/Veritcal pane      |
| `Ctrl + Shift + Up/Down`         | Previous/Next pane            |
| `Ctrl + b + s`                   | Toggle Synchronize panes      |

###### Nested Management
| Key                              | Description                   |
| -------------------------------- | ----------------------------- |
| `Ctrl + p`                       | Toggle LOCAL/REMOTE mode      |
